DTNT FAQ

Answers for the install, update, and Pro questions customers ask first.

This FAQ keeps the commercial path simple. It covers the practical questions around Windows support, Store delivery, Pro activation, and what to expect when using DTNT in the field.

PlatformWindows 10 and 11
Preferred installMicrosoft Store
Commercial modelFree core plus one-time Pro unlock
Support pathDelivery email plus in-app activation

Does DTNT work on Windows only?

Yes. DTNT is a Windows desktop app built for Windows 10 and Windows 11.

Should I install DTNT from Microsoft Store or the website?

Microsoft Store is the preferred install path for the approved Windows build. If Store access is blocked on a machine, contact Denali Tech support for the best backup path.

Can I use DTNT before buying Pro?

Yes. Free lets you use the main field workflows first: Wi-Fi survey, live channel view, IP scanning, remembered hosts, labels, filtering, Copy IP, speed tests, saved in-app speed history, saved profile review, and ping-once checks.

How does Pro activation work?

Buy Pro through the DTNT checkout, then use the purchase email and activation code from the delivery email inside the Profiles workspace.

What does DTNT stand for?

DTNT stands for Denali Tech Net-Tools, the Windows Wi-Fi and network diagnostics app from Denali Tech.

What if Microsoft Store is blocked on a machine?

Use the Microsoft Store path first. If Store access is blocked, contact Denali Tech support with the machine and order context so we can recommend the right backup path.

What does Pro unlock?

Pro unlocks best-channel guidance, richer lane-pressure detail, selected-host browser follow-up, port checks, Copy Name and MAC, exports, saved Wi-Fi key reveal, profile removal, continuous ping, trace route, and the commercial activation path.

Do I need Pro just to test the app?

No. Free exists so you can install DTNT, verify it works on the machine, and use the main product surfaces before deciding whether Pro is worth it for your workflow.

Why are install and Pro purchase separate?

Keeping install separate from the paid unlock makes the product easier to try, easier to deploy, and easier to support without forcing the same path on every machine.
