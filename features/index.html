What the app covers

Use one shell instead of stitching together separate tools.

These are the product surfaces DTNT is built around today. The goal is not more tabs for the sake of more tabs. It is tighter workflows for Wi-Fi inspection, local-device discovery, selected-host checks, and repeatable proof when you need to show what happened on a network.

Wi-Fi analyzer

Free: review nearby radios, signal posture, current SSID, band, channel, and security from a readable live survey surface.

Channel analysis

Free: live channel view. Pro: best-channel guidance, richer lane-pressure detail, competing-radio context, and clearer deployment calls.

IP scanner

Free: refresh or fresh scan, remembered online/offline hosts, labels, filtering, web indicators, and Copy IP. Pro: open device pages, port checks, Copy Name/MAC, and export.

Speed test

Free: run speed tests and keep saved in-app history. Pro: export speed history to JSON or CSV when you need handoff evidence.

Profiles

Free: review saved Wi-Fi profile names, security, and live-versus-saved comparison. Pro: reveal/copy saved keys, remove profiles, and export.

Diagnostics

Free: ping once from the Tools workspace. Pro: continuous ping and trace route for longer troubleshooting sessions.

Field techs, installers, owners, and anyone who needs one steady Windows network utility.
