# DTNT low-budget LinkedIn launch plan

Date: 2026-05-09

## Goal

Sell the first DTNT Pro license, or at minimum learn which field-tech audience is willing to click through.

## Current lesson from the earlier boost

- Spend seen in screenshot: $44 of $50
- Video views seen in screenshot: 999
- Cost per video view: $0.04
- Audience skew seen in screenshot:
  - 70% entry level
  - 34% at 10,001+ employee companies
  - 22% IT Services and IT Consulting

That is fine for reach, but weak for first-license intent. DTNT is more likely to sell first to working techs, lead installers, owners, and small integrators than entry-level viewers at very large companies.

https://dt-nt.com/field-tech/?utm_source=linkedin&utm_medium=paid_social&utm_campaign=field_tech_launch&utm_content=boost_post

- Wi-Fi analysis
- IP scanning
- web targets
- IP scanning, remembered hosts, labels, filters, and Copy IP
- speed tests and saved in-app speed history
- Pro selected-host port checks and exports
- Microsoft Store install
- $59 launch Pro

## Recommended boost setup

Objective:

Website visits

Why:

LinkedIn maps Website visits to landing-page clicks. Video views and impressions are cheaper but weaker if the goal is a license sale.

Budget:

- Test 1: $20 total for 2 days
- If link clicks are happening: add $30 more
- If no meaningful clicks after $20: stop and change creative/message

Do not spend the full $50 blindly again.

## Kill rules

After $20:

- If link clicks are under 8: stop and change the hook or targeting
- If link clicks are over 8 but no buy clicks: improve landing page CTA or offer
- If buy clicks happen but no purchase: check checkout friction and activation trust language
- If one sale happens: do not change everything; add $30 to the same audience and message
